# tclaw — 下一代 OpenClaw

## 项目定位

借鉴 OpenClaw 的设计思想，在关键选型上做自己的决策，自建体系。

目标：构建一套更轻量、更 Pythonic 的 AI Agent 网关与后端。

## MVP 阶段目标

修复和完善网关与后端核心功能，打下可运行的基础。

## 核心约定

- **语言**：Python 3.11+
- **架构哲学**：借鉴 OpenClaw 的设计思想，但自建插件/Skill 体系，不依赖 OpenClaw 生态
- **代码规范**：(待定)
- **文件结构**：

```
tclaw/
├── src/
│   └── tclaw/
│       ├── __init__.py
│       ├── gateway/      # 网关核心
│       ├── backend/      # 后端核心
│       └── common/       # 公共工具/类型
├── pyproject.toml
├── README.md
└── .gitignore
```
